Simple Reinforcement Learning Showcase

This is a simple showcase of how reinforcement learning can be used to solve the travelling saleman problem (TSP). In essence, the TSP is about finding the shortest path between a range of locations. In this showcase, the TSP has being adapted to finding the shortest path between offshore wind turbines in an environment. Although simple, it's a taste of how AI, especially reinforcement learning, can be utilised to solve maintenance problems.
